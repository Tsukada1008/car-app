# car-app

Simple single-file SPA for managing loan cars and dealer numbers.

Open `index.html` in a browser to run locally.

## How to push to GitHub
See the instructions in the companion documentation or run these commands in PowerShell:

```powershell
# Set location to project
cd "C:\Users\Tsukada\car-app"
# Add files and commit
git add .
git commit -m "Add README and .gitignore"
```

Then add remote and push (see remote setup instructions).
